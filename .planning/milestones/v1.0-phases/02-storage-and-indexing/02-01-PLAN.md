Create `src/services/crawler.ts` with the following implementation:

1. Import `fast-glob` (default import as `fg`), `ignore` (default import), `readFile` from `node:fs/promises`, `childLogger` from `./logger.js`.
2. Create `const log = childLogger('crawler');`
3. Export `SOURCE_EXTENSIONS` as a `Set<string>` containing: `.ts`, `.tsx`, `.mts`, `.cts`, `.js`, `.jsx`, `.mjs`, `.cjs`, `.py`, `.pyi`, `.go`, `.rs`
4. Export `ALWAYS_EXCLUDE_GLOBS` as a `string[]` containing: `**/node_modules/**`, `**/.git/**`, `**/dist/**`, `**/build/**`, `**/.next/**`, `**/__pycache__/**`, `**/*.egg-info/**`, `**/package-lock.json`, `**/yarn.lock`, `**/pnpm-lock.yaml`, `**/Cargo.lock`, `**/*.min.js`
5. Export `async function crawlSourceFiles(rootDir: string): Promise<string[]>` that:
   a. Creates an `ignore()` instance
   b. Tries to read `${rootDir}/.gitignore` and add its content to the ignore instance (catch and skip if no .gitignore)
   c. Calls `fg('**/*', { cwd: rootDir, absolute: true, ignore: ALWAYS_EXCLUDE_GLOBS, onlyFiles: true })`
   d. Filters results: keep only files whose extension is in `SOURCE_EXTENSIONS` AND not ignored by the gitignore instance (use relative path from rootDir for ignore check)
   e. Logs the count at info level: `log.info({ rootDir, fileCount: result.length }, 'Crawl complete');`
   f. Returns the filtered array
