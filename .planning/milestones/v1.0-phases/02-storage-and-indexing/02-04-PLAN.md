Export `async function runIndex(targetPath?: string): Promise<void>`:

1. **Resolve path**: `const rootDir = resolve(targetPath ?? '.');`

2. **Read profile**: Call `readProfile()`. If null, print error to stderr: `"No profile found. Run 'brain-cache init' first.\n"` and `process.exit(1)`.

3. **Check Ollama**: Call `isOllamaRunning()`. If false, print error to stderr: `"Error: Ollama is not running. Start it with 'ollama serve' or run 'brain-cache init'.\n"` and `process.exit(1)`.

4. **Determine dimensions**: Look up `EMBEDDING_DIMENSIONS[profile.embeddingModel]`. If undefined, use 768 as default and log warning.

5. **Open LanceDB**: Call `openDatabase(rootDir)` then `openOrCreateChunkTable(db, profile.embeddingModel, dim)`.

6. **Crawl**: Call `crawlSourceFiles(rootDir)`. Print to stderr: `"brain-cache: found ${files.length} source files\n"`. If 0 files, print `"No source files found in ${rootDir}\n"` and return (not exit â€” this is not an error).

7. **Chunk all files**: For each file:
   a. Read content with `readFile(filePath, 'utf-8')`
   b. Call `chunkFile(filePath, content)`
   c. Collect all chunks into a flat array
   d. Print progress to stderr every 10 files: `"brain-cache: chunked ${i}/${files.length} files\n"`
   Print total: `"brain-cache: ${allChunks.length} chunks from ${files.length} files\n"`

8. **Batch embed + store**: Process chunks in batches of `DEFAULT_BATCH_SIZE`:
   a. For each batch, extract `.content` from each chunk
   b. Call `embedBatchWithRetry(profile.embeddingModel, texts)`
   c. Map chunks + vectors to `ChunkRow[]`:
      ```typescript
      const rows: ChunkRow[] = batch.map((chunk, i) => ({
        id: chunk.id,
        file_path: chunk.filePath,
        chunk_type: chunk.chunkType,
        scope: chunk.scope,
        name: chunk.name,
        content: chunk.content,
        start_line: chunk.startLine,
        end_line: chunk.endLine,
        vector: vectors[i],
      }));
      ```
   d. Call `insertChunks(table, rows)`
   e. Print progress to stderr every batch: `"brain-cache: embedded ${processedCount}/${allChunks.length} chunks\n"`

9. **Write index state**: Call `writeIndexState(rootDir, { version: 1, embeddingModel: profile.embeddingModel, dimension: dim, indexedAt: new Date().toISOString(), fileCount: files.length, chunkCount: allChunks.length })`.

10. **Print summary**: `"brain-cache: indexing complete\n  Files: ${files.length}\n  Chunks: ${allChunks.length}\n  Model: ${profile.embeddingModel}\n  Stored in: ${rootDir}/.brain-cache/\n"`

Follow the same patterns as `runInit`:
- All output via `process.stderr.write()` â€” zero stdout
- `process.exit(1)` for fatal errors (no profile, no Ollama)
- Non-fatal issues (0 files) just return
