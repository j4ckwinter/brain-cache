       - Define internal `ChunkRow` interface matching LanceDB snake_case columns: `{ id, file_path, chunk_type, scope, name, content, start_line, end_line, [key: string]: unknown }`
       - Export `resolveSymbolToChunkId(chunksTable: Table, toSymbol: string, fromFile: string): Promise<string | null>`:
         - Escape `toSymbol` with `.replace(/'/g, "''")`
         - Query: `chunksTable.query().where(\`name = '${escaped}'\`).toArray()`

         - `visited = new Set<string>()`
         - `queue: Array<{ chunkId: string; depth: number }> = [{ chunkId: seedChunkId, depth: 0 }]`
         - `hops: FlowHop[] = []`
         - BFS loop:
           - Shift from queue, skip if visited, add to visited
           - Fetch chunk row: `chunksTable.query().where(\`id = '${escaped}'\`).toArray()`
           - If no row found, continue (skip)
           - Push to hops: `{ chunkId, filePath: row.file_path, name: row.name, startLine: row.start_line, endLine: row.end_line, content: row.content, hopDepth: depth }`
           - If `depth >= maxHops`, continue (don't enqueue children)
           - Get outgoing edges: `queryEdgesFrom(edgesTable, chunkId)`
           - Filter to call edges only: `edges.filter(e => e.edge_type === 'call')`
           - For each call edge, resolve toSymbol → chunkId via `resolveSymbolToChunkId`
           - If resolved and not visited, push to queue with `depth + 1`
       - Log at debug level: seed chunk, hops found, max depth reached
