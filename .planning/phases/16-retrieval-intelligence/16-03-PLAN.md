    - groupChunksByFile sorts chunks within each file group by startLine ascending
    - groupChunksByFile with empty input returns empty Map
    - groupChunksByFile with chunks from 3 files returns Map with 3 entries
    - enrichWithParentClass adds parent class chunk when method chunk has scope and parent class exists in chunks table
    - enrichWithParentClass does NOT add parent if it would exceed token budget
    - enrichWithParentClass does NOT add parent if parent is already in the chunk set
    - enrichWithParentClass skips non-method chunks (function, class, file types)
    - formatGroupedContext outputs file-header sections separated by ---
    - formatGroupedContext file header format: "// ── {filePath} ──"
  </behavior>
  <action>
    Create `src/services/cohesion.ts`:

    1. Import `RetrievedChunk` from `../lib/types.js`, `formatChunk`, `countChunkTokens` from `./tokenCounter.js`, `childLogger` from `./logger.js`, `Table` from `@lancedb/lancedb`.

    2. Export `groupChunksByFile(chunks: RetrievedChunk[]): Map<string, RetrievedChunk[]>`:
       - Create `Map<string, RetrievedChunk[]>`
       - For each chunk, push to its filePath group
       - Sort each group by `startLine` ascending
       - Return the map

    3. Export `enrichWithParentClass(chunks: RetrievedChunk[], chunksTable: Table, opts: { maxTokens: number; currentTokens: number }): Promise<RetrievedChunk[]>`:
       - Create a Set of existing chunk IDs for O(1) lookup
       - For each chunk where `chunkType === 'method'` and `scope !== null`:
         - Escape scope: `scope.replace(/'/g, "''")`
         - Query: `chunksTable.query().where(\`name = '${escaped}' AND file_path = '${escapedFilePath}' AND chunk_type = 'class'\`).toArray()`
         - If found and not already in the set:
           - Calculate token cost: `countChunkTokens(formatChunk(parentAsRetrievedChunk))`
           - If `opts.currentTokens + tokenCost <= opts.maxTokens`: add to result, update currentTokens, add to seen set
       - Return enriched chunks array (original chunks + added parents, parents prepended before their methods in the array)

    4. Export `formatGroupedContext(groups: Map<string, RetrievedChunk[]>): string`:
       - For each `[filePath, chunks]` entry:
         - Header: `// ── ${filePath} ──`
         - Body: `chunks.map(formatChunk).join('\n\n')`
         - Section: `${header}\n${body}`
       - Join sections with `'\n\n---\n\n'`

    Create `tests/services/cohesion.test.ts`:
    - Mock `@lancedb/lancedb`, `../../src/services/logger.js`
    - Mock `../../src/services/tokenCounter.js` with `formatChunk` returning `// File: ... \n content` and `countChunkTokens` returning a fixed value (e.g., 10)
    - Create helper `makeChunk(overrides)` returning a `RetrievedChunk`
    - Test all behaviors listed above
    - For `enrichWithParentClass`: mock chunksTable with `.query().where().toArray()` chain
