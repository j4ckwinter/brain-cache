  <action>
    Create `src/workflows/watch.ts` exporting `runWatch(targetPath?: string): Promise<void>`:

    3. Import `createWatcher` from `../services/fileWatcher.js`.
    4. Import `runIndex` from `./index.js`.

    **Cross-process lock functions** (export for testing):
    - `acquireIndexLock(projectRoot: string): Promise<boolean>`:
      - `lockPath = join(projectRoot, '.brain-cache', '.indexing')`
      - Ensure `.brain-cache` dir exists with `mkdir(join(projectRoot, '.brain-cache'), { recursive: true })`
      - Use `open(lockPath, 'wx')` (O_EXCL flag ŌĆö atomic create, fails if exists)
      - On success: close the file handle, return `true`
      - On error: return `false`
    - `releaseIndexLock(projectRoot: string): Promise<void>`:
      - `unlink(join(projectRoot, '.brain-cache', '.indexing')).catch(() => undefined)`

    **Debounce + accumulation** (module-level state, exported for testing):
    - `let debounceTimer: ReturnType<typeof setTimeout> | null = null`
    - `const pendingFiles = new Set<string>()`
    - `scheduleReindex(filePath: string, eventType: string, projectRoot: string): void`
      - Write to stderr: `brain-cache: ${eventType} ${filePath}\n`
      - `pendingFiles.add(filePath)`
      - If `debounceTimer !== null`, call `clearTimeout(debounceTimer)`
      - Set `debounceTimer = setTimeout(async () => { ... }, 500)`
      - Inside the timeout callback:
        - `debounceTimer = null`
        - `const files = [...pendingFiles]; pendingFiles.clear()`
        - Call `acquireIndexLock(projectRoot)` ŌĆö if false, write `brain-cache: skipping reindex ŌĆö another index process is running\n` to stderr, return
        - `try { await runIndex(projectRoot) } finally { await releaseIndexLock(projectRoot) }`

    **runWatch orchestrator:**
    - `const rootDir = resolve(targetPath ?? '.')`
    - Write to stderr: `brain-cache: watching ${rootDir}\n`
    - `const watcher = await createWatcher(rootDir)`
    - Register events: `add`, `change`, `unlink`, `addDir`, `unlinkDir` all call `scheduleReindex(fp, eventName, rootDir)`
    - Register `error` event: write error to stderr
    - Register `ready` event: write `brain-cache: watcher ready\n` to stderr
    - Cleanup function:
      - `if (debounceTimer !== null) clearTimeout(debounceTimer)`
      - `pendingFiles.clear()`
      - `await watcher.close()`
      - `process.exit(0)`
    - `process.once('SIGINT', cleanup)`
    - `process.once('SIGTERM', cleanup)`
    - `await new Promise(() => {})` ŌĆö keep process alive
