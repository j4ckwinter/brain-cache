    **Step 2: Create src/services/sessionStats.ts**

    The service exports:
    - `SESSION_STATS_PATH` — `join(GLOBAL_CONFIG_DIR, SESSION_STATS_FILENAME)`
    - `STATS_TTL_MS` — `2 * 60 * 60 * 1000` (2 hours)
    - `interface StatsDelta { tokensSent: number; estimatedWithoutBraincache: number; }`
    - `interface SessionStats { tokensSent: number; estimatedWithoutBraincache: number; callCount: number; lastUpdatedAt: string; }`
    - `function accumulateStats(delta: StatsDelta, ttlMs?: number): Promise<void>`

    Implementation details:
    - Module-level mutex: `let _statsMutex: Promise<void> = Promise.resolve();`
    - `accumulateStats` chains onto `_statsMutex` exactly like `withWriteLock` in lancedb.ts — the `then(() => undefined, () => undefined)` pattern prevents deadlock
    - Internal `_doAccumulate(delta, ttlMs)`:
      1. `await mkdir(GLOBAL_CONFIG_DIR, { recursive: true })`
      2. Read existing stats file via `_readStats()` — returns `SessionStats | null`, catches all errors and returns null
      3. TTL check: `const isExpired = existing === null || (Date.now() - Date.parse(existing.lastUpdatedAt)) > ttlMs`
      4. If expired: base = `{ tokensSent: 0, estimatedWithoutBraincache: 0, callCount: 0 }`; else base = existing
      5. Compute updated: `{ tokensSent: base.tokensSent + delta.tokensSent, estimatedWithoutBraincache: base.estimatedWithoutBraincache + delta.estimatedWithoutBraincache, callCount: base.callCount + 1, lastUpdatedAt: new Date(now).toISOString() }`
      6. Atomic write: `await writeFile(tmpPath, JSON.stringify(updated, null, 2), 'utf-8'); await rename(tmpPath, SESSION_STATS_PATH);` where `tmpPath = SESSION_STATS_PATH + '.tmp'`
    - Import `readFile, writeFile, rename, mkdir` from `node:fs/promises`
    - Import `join` from `node:path`
    - Import `GLOBAL_CONFIG_DIR, SESSION_STATS_FILENAME` from `../lib/config.js`
    - Import `childLogger` from `./logger.js`
    - Log at warn level on failure: `log.warn({ err }, 'stats accumulation failed')`

    Also export a `_resetMutexForTesting()` function that resets the mutex to `Promise.resolve()` — this is needed for test isolation. Guard with a comment `// @internal — test use only`.
